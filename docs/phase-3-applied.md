# Fase 3 Aplicada

Aplicación inicial del prompt para Noxis.

## Criterio de prioridad

Puntaje base sugerido para ranking inicial:
- `seoPriority`: demanda de búsqueda y potencial de long-tail.
- `monetizationPriority`: probabilidad de afiliación, upsell o lead gen.
- `productValue`: utilidad real para el usuario final.
- `availability`: claridad de pricing, planes o acceso público.

Ranking global sugerido: `0.35 * seoPriority + 0.30 * monetizationPriority + 0.20 * productValue + 0.15 * availability`.
